# Thuto AI

Educational support platform for South African students, combining a Vue.js frontend with a Node.js backend.

## Tech Stack

- **Frontend:** Vue.js, HTML/CSS/JS
- **Backend:** Node.js, Express
- **Mobile:** React Native (Expo)

## Features

- AI-assisted learning tools
- Student resource management
- Cross-platform (web + mobile)
- South African curriculum-aligned content

## Getting Started

```bash
git clone https://github.com/Raphasha27/thuto-ai.git
cd thuto-ai
# Server
cd server && npm install && npm start
# Client
cd client && npm install && npm run dev
```
